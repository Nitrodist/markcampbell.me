Jeffrey Zeldman's Designing with Web Standards Notes

The following document are the notes that I wrote while reading this excellent book. Please note that these notes have not been edited for content in any way. Alternatively, you can view the raw text file that I had created here.

Information about this book can be viewed here on Zeldman's website and here on Wikipedia. Zeldman's website can be found here at zeldman.com.

If you have any questions, concerns, comments, etc., with these notes, please contact me on my Github account (github.com/Nitrodist).
